####In version 0.1

``quantarhei.DFunction``
* If x-axis has units management (TimeAxis, FrequencyAxis), label is created automatically

``quantarhei.CorrelationFunction``
* x-axis controlled by ``time_units`` context manager, y-axis by ``energy_units`` context manager

``quantarhei.SpectralDensity``
* both axes are controlled by ``energy_units`` context manager; one may specify y-axis units independently

``quantarhei.PlottableCollection``
* collection of plottable objects to be plotted in one plot

``quantarhei.AbsSpect``
* x-axis controlled by ``energy_units`` context manager; y-axis in default units

``quantarhei.AbsCollection``
* collection of AbsSpect objects to be plotted in one plot
